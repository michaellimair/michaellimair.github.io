Positioning Schemes Used

Aside from the static (normal) flow, here are several positioning schemes used for completing this task:

  1. Absolute Positioning
  2. Used to make object positioning comply with its parent container. One of its uses is vertical alignment of a div inside a parent div. The usage of absolute positioning can be found here:
    • Video Overlay: see css/index.css [line 29, 190]
    • Order Block: see css/index.css [line 117]
    • Form Legend: see css/order.css [line 51]
    • Cake Image: see css/order.css [line 105]
  3. Fixed Positioning
  4. Fixed layout is used because the objects should not go away from the viewport when the user is scrolling on the webpage or moving the viewport in any way. Therefore, it must be fixed to a place using the CSS property "position: fixed;" The usage of fixed positioning can be found here:

    • Navigation Button Container: see css/master.css [line 227]
    • Navigation Menu Icon (CSS Drawing): see css/master.css [line 284]

  6. Relative positioning makes the positioning of blocks adherent to normal flow even though it is contained in a block. CSS Stylesheet Usage

For this project, no inline or embedded CSS is used, because using those CSS insertion methods might complicate things as it is difficult to edit the CSS properties when needed. For every page requiring CSS styling (Home, Order, and Design pages), three stylesheets are linked to it externally:

  1. css/reset.css used to reset browser defaults, ensuring uniformity of styling between different browsers
  2. css/master.css used to apply the same general styling shared by all pages, ensuring a uniform look across pages and to avoid CSS code duplication, which makes editing the CSS styling complicated and tedious.
  3. Styling of individual pages (css/index.css, css/order.css, or css/print.css) to make changes to styling for elements unique to each page.

@media query is also included in the master and indivudial pages' CSS stylesheet so that different devices will display the website differently according to the needs (e.g. mobile phone, computer, tablet, printer, etc.).
